## Why These Docs Exist

These documents are meant to keep the project focused while development is still early.

They define:

- what Ascension Analytics is supposed to do
- what is intentionally out of scope
- how data should move through the system
- how we should sequence the build

The shared analytics spec remains the source of truth for the event contract. The other docs explain how we use that contract to build the product.
